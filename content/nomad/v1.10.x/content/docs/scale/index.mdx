# Scale Nomad

This section provides information on the following standalone products:

- Nomad Autoscaler, a horizontal application and cluster autoscaler
  for Nomad.
- Nomad Bench, a project that provides reusable infrastructure automation to
run test scenarios in order to collect metrics and data from Nomad clusters
running at scale.
