# `trim` Function

`trim` removes the specified characters from the start and end of the given
string.

## Examples

```shell-session
> trim("?!hello?!", "!?")
hello
```

## Related Functions

- [`trimprefix`](/nomad/docs/job-specification/hcl2/functions/string/trimprefix) removes a word from the start of a string.
- [`trimsuffix`](/nomad/docs/job-specification/hcl2/functions/string/trimsuffix) removes a word from the end of a string.
- [`trimspace`](/nomad/docs/job-specification/hcl2/functions/string/trimspace) removes all types of whitespace from
  both the start and the end of a string.
